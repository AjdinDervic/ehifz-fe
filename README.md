# Frontend setup

## 1. Clone repository
```bash
git clone https://github.com/AjdinDervic/ehifz-fe
cd ehifz-fe
```
## 2. Install dependencies
```bash
npm install
```

## 3. Start development server
```bash
npm run dev
```

Frontend will be available at:
http://localhost:5173
